    # IA Assistance

Durante el desarrollo de este challenge se utilizó asistencia mediante un Large Language Model (LLM) como herramienta de apoyo técnico.

## Alcance del uso

La asistencia fue utilizada para:

- Aclarar dudas sobre la interpretación de los requerimientos y el alcance del challenge.
- Revisar expresiones  y transformaciones en PySpark.
- Proponer mejoras en la organización/estructura del proyecto.
- Colaborar en la redacción del README y documentación técnica.
- Detectar posibles errores de implementación y sugerir alternativas.
- Resolver dudas sobre el enunciado.

## Criterio de aceptación

Ninguna propuesta generada por el LLM fue incorporada automáticamente.

Cada sugerencia fue evaluada verificando:

- Compatibilidad con Apache Spark y Databricks.
- Correcto funcionamiento mediante ejecución del código.
- Consistencia con los requerimientos del challenge.
- Validación de resultados utilizando consultas y controles sobre las tablas Delta generadas.

Las decisiones finales de implementación, adaptación y validación fueron realizadas manualmente.


## Resultado

El LLM fue utilizado como herramienta de apoyo para acelerar el desarrollo y la documentación del proyecto, manteniendo siempre una validación manual antes de aceptar cualquier propuesta.
